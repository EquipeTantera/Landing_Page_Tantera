## 📁 Estrutura de pastas

|--> docs<br>
|--> src<br>
  &emsp;| --> backend <br>
  &emsp;|--> frontend<br>
| readme.md<br>

- **docs**: pasta que contém toda a documentação do projeto
- **src:** pasta que contém todo o código fonte do projeto
- **frontend:** pasta que contém o código fonte do frontend do projeto
